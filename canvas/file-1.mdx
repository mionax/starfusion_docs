文字描述直接生成视频内容，用于快速表达创意、生成动态画面或完成从想法到视频的初步制作。

## 通过文字生成视频

<video src="/videos/文生图-自由画板-1.mp4" controls />

### 使用方式：

1. 输入你想生成的视频内容描述
2. 选择合适的生成模型
3. 确认视频时长、比例、分辨率参数
4. 执行生成

建议尽量描述清楚主体、动作、场景和镜头感，以获得更稳定的结果。

### 适合场景

文字生成视频适合快速从想法生成动态内容，常见场景包括：

- 产品展示
- 广告短片
- 创意概念视频
- 社交媒体内容
- 分镜或动态草稿生成

### 提示词建议

为了获得更符合预期的视频结果，建议在描述中尽量包含以下信息：

- 主体：视频中谁或什么是重点
- 动作：主体正在做什么
- 场景：发生在什么环境中
- 风格：写实、电影感、科技感、商业广告风格等
- 镜头感：近景、远景、推进、跟随、俯拍等

描述越明确，生成结果通常越稳定。
