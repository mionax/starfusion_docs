# 左侧工具栏

左侧工具栏聚合了画布中最常用的创作入口。

<Frame>
  ![Image](/images/image-33.png)
</Frame>

## 上传图片

用于从你的设备导入图片到画布。

使用方式：

1. 点击上传入口
2. 选择设备中的图片，或将图片拖拽到画布
3. 图片会作为卡片出现在画布中

支持常见图片格式，例如 JPG、PNG。

## 评论

用于在画布中添加沟通记录、修改建议或创作备注。

适合团队协作和复盘时使用。

## 角度编辑

角度编辑可以基于选中的图片创建多角度图像，并尽量保持主体与背景一致。

使用方式：

1. 选中需要编辑的图片
2. 打开角度编辑入口
3. 切换角度或输入提示词
4. 生成新的角度结果

<video src="/videos/多角度编辑2-1.mp4" controls />

## 文本

用于在画布中创建文本内容，记录文案、说明或创作备注。

使用方式：

1. 点击文本工具
2. 在画布上点击放置文本框
3. 输入内容

## 模板库

模板库用于浏览预设模板和参考场景，帮助你更快开始创作。
