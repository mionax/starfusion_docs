# 图像生成与编辑

Draftio 支持从文字生成图像，也支持基于已有图片继续编辑。

<video src="/videos/文生图.mp4" controls />

## 通过提示词生成图像

适合从零开始创建视觉内容。

建议提示词包含：

- 主体
- 风格
- 场景
- 光线
- 构图
- 用途

提示词越具体，结果越容易接近期望。

## 通过提示词编辑图像

适合对已有图片进行修改、替换、优化或局部调整。

使用方式：

1. 选中需要编辑的图片
2. 输入修改目标
3. 选择合适的生成能力和画面设置
4. 执行编辑

<video src="/videos/多图关联.mp4" controls />

## 多图参考编辑

多图参考编辑可以把多张图中的元素、风格或结构结合到新的结果中。

适合场景：

- 服装试穿
- 商品合成
- 物品迁移
- 风格统一
- 角色或主体保持一致

建议选择主体清晰、背景简洁的图片作为参考。

<video src="/videos/每张图参与下次创作.mp4" controls />

## 图片预览页高级编辑

图片预览页适合做更细粒度处理：

- 精准编辑：点击局部区域后进行细节修改
- 消除元素：移除多余物体、水印或瑕疵
- 融合：跨图搬运物体并合成到新场景

这些能力更适合在一张图接近成品时使用。

详见 [高级图像编辑](./advanced-image-editing)。
