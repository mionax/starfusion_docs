# Draftio Docs

Draftio 是一款面向创作者的 AI 可视化创作画布。这里汇总上手路径、画布操作、AI 能力、计费规则和常见问题，帮助你更快从想法进入成品。

## 从这里开始

- 第一次使用：阅读 [快速开始](./getting-started/quickstart)
- 想了解画布界面：阅读 [创作画布概览](./canvas/overview)
- 想生成图片或视频：阅读 [输入框与生成设置](./canvas/prompt-input)
- 想处理生成结果：阅读 [图像卡片操作](./canvas/image-cards)
- 想了解不同生成能力：阅读 [图片生成能力](./models/image-models) 和 [视频生成能力](./models/video-models)

## 这套文档的路径

- `开始使用`：注册、进入工作区、创建第一个项目
- `创作画布`：界面结构、工具栏、输入框、图像卡片和快捷键
- `AI 能力`：图片生成与视频生成的差异和适用场景
- `电商工具`：一键生成商品详情图等专项能力
- `计费与账户`：套餐、积分、商用和使用条件相关说明
- `支持`：常见问题和排查入口
